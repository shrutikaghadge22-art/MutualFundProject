# MutualFundProject
## Project Overview
This project analyzes mutual fund performance, investor participation, risk metrics, and category trends using Python and Power BI.

## Features
- Data Ingestion Pipeline
- Exploratory Data Analysis (EDA)
- NAV Trend Analysis
- Rolling Sharpe Ratio Analysis
- Risk and Return Analysis
- Mutual Fund Recommendation System
- Interactive Power BI Dashboard

## Dataset Information
The dataset contains:
- Fund House Information
- Scheme Details
- NAV Values
- SIP Inflows
- Assets Under Management (AUM)
- Investor Folios
- Historical Returns
- Fund Categories

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- SQLite
- Power BI
- GitHub

## How to Run
1. Clone the repository
2. Install required libraries
3. Run data_ingestion.py
4. Run analytics notebooks
5. Open Power BI dashboard
